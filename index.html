參數格式／選項說明

padding0–6儲存格內縮空格數

cell_pad_right0–8儲存格右側空白數

自訂框線模板 

                        

                        

                            
grid_config 格式說明

                            
九宮格區塊（7–8–9／4–5–6／1–2–3）：

                            

                                
title標題框{title} 置中；未定義列跳過

                                
top表格頂線無 title 時使用

                                
bottom表格底線1,2,3 或預設

                                
footer頁尾框{footer} 置中；未定義列跳過

                                
cells.5資料儲存格格式（僅 ASCII）CSS/PIL 未來實作。Python format："  {}" 左空 2 格；"  {:.2f}" 小數兩位

                            

                            
分隔線（title_sep、header_sep）位置 4,5,6,0：

                            

                                
值效果

                                
{"4":"├","5":"─","6":"┤","0":"┼"}自訂字元

                                
null 或刪除用 style 預設

                                
false不畫該分隔線

                                
只設 4,5,6 未設 0交叉處為空 → ├───┤

                            

                            
其他區塊（title, top, bottom, footer）：

                            
無 null/false 語意。未定義的列會被跳過；定義的列會套用，缺鍵使用 style 預設。僅 title_sep、header_sep 支援 null 與 false。

                            
padding / 儲存格格式：

                            
每欄寬度 = max(內容 display width) + padding×2。左空 2 格可用 cells.5: "  {}" 或 cell_pad_left=2。
cells.5 格式範例（Python format，: 後為格式規格）：

                            

                                
"  {:.2f}"小數兩位"  {:>10}"右對齊寬 10

                                
"{:,.2f}"千分位、小數兩位"{:.1%}"百分比一位

                            

                        

                        

                        
校準 (校準輸出字元寬度)

                        
以 ASCII 寬度 1.0 為基準，動態調整校準輸出 Render 結果。留空則使用後端預設 (wcwidth)。
僅 Render Backend 會套用；Frontend 預覽不支援。
